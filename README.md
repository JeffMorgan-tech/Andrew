## How to view locally

Open `index.html` in any modern browser, or serve the folder with any static file server:

```
python -m http.server 8000
# then visit http://localhost:8000
```

## Deployment

This is a fully static site — no build step required. Deploy the entire folder contents to any static host (Vercel, Netlify, Cloudflare Pages, GitHub Pages, etc.).
